# Memory Bank: Product Context

## Product Description
Демонстрационное приложение, показывающее, как лениво подгружать компоненты в React с помощью `React.lazy()` и `Suspense`, управляя списком динамически добавляемых компонентов.

## User Needs
- Понять, как реализовать ленивую загрузку компонентов в небольшом, изолированном примере.
- Увидеть рабочий пример с типобезопасностью (TypeScript) и понятной архитектурой.

## Business Context
- Учебный микропроект в составе репозитория `learn-react`.
- Используется как база для экспериментов с lazy loading и расширения на новые подходы (динамический `import()`, `@loadable/component`, Module Federation и т.п.).
