# Memory Bank: Project Brief

## Project Overview
Демонстрационный проект на React + TypeScript для показа возможности динамически подгружаемых компонентов в React.

Проект демонстрирует паттерн lazy loading компонентов с использованием React.lazy() и Suspense API.

## Goals
1. Создать демонстрационное приложение с динамической загрузкой компонентов
2. Показать работу React.lazy() и Suspense для code splitting
3. Реализовать систему управления состоянием загруженных компонентов
4. Обеспечить типобезопасность через TypeScript

## Scope
### Включено:
- Основное приложение с двумя кнопками управления
- Два демонстрационных компонента (Component1, Component2)
- Система динамической загрузки компонентов
- Контейнер для отображения загруженных компонентов
- TypeScript типизация всех компонентов

### Не включено:
- Серверная часть
- База данных
- Аутентификация
- Сложная бизнес-логика
- Тестирование (на данном этапе)
