## Общее описание проекта

Репозиторий `learn-react` предназначен для изучения React и содержит множество микропроектов.

## Интеграция с Cursor Memory Bank

- IDE: Cursor
- Система: Memory Bank System v0.8 от `vanzan01` (см. `https://github.com/vanzan01/cursor-memory-bank.git`)
- Раздел для индексации: секция `<microprojects-list>` в файле `readme.md`

## Микропроекты (по данным из `readme.md`)

- **Кодовое имя**: `react-lazy`  
  - DIRECTORY: `lazy-load-components/react-lazy`  
  - DESCRIPTION: ленивая загрузка компонент при помощи `React.lazy`

## Замечания

- Секция `<microprojects-groups>` предназначена только для пользователей и **не используется** системой `cursor-memory-bank`.
